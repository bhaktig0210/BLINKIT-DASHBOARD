# BLINKIT-DASHBOARD

# 🛒 Blinkit Grocery Sales Dashboard | Power BI

## 📌 Project Overview

The **Blinkit Grocery Sales Dashboard** is an interactive Business Intelligence project developed using **Power BI** to analyze grocery sales performance and customer purchasing trends. The dashboard provides valuable insights into sales, product performance, outlet performance, and customer preferences through dynamic visualizations and KPI metrics.

This project demonstrates skills in data cleaning, data modeling, DAX, and dashboard development to support data-driven business decisions.

---

# 🎯 Objectives

- Analyze overall grocery sales performance.
- Monitor key business KPIs.
- Identify top-performing product categories.
- Compare outlet performance across different locations and sizes.
- Analyze customer purchasing patterns.
- Build an interactive dashboard for business reporting.

---

# 📂 Dataset Information

The project uses a grocery sales dataset containing transactional and outlet information.

### Dataset includes:

- Item Identifier
- Item Type
- Fat Content
- Item Weight
- Outlet Identifier
- Outlet Size
- Outlet Location Type
- Outlet Type
- Outlet Establishment Year
- Sales
- Rating
- Visibility

**Dataset Format:** Excel (.xlsx)

# ▶️ How to Run the Project

1. Clone this repository.

```bash
git clone https://github.com/bhaktig0210/Blinkit-Grocery-Sales-Dashboard.git
```

2. Open **Power BI Desktop**.

3. Open the `BLINKIT DASHBOARD.pbix` file.

4. If prompted, reconnect the dataset (`BlinkIT Grocery Data.xlsx`).

5. Refresh the data.

6. Explore the dashboard using interactive filters and slicers.
